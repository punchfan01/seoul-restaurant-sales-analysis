# 서울시 음식점 매출 영향 요소 분석

서울시 골목상권 데이터를 활용하여 음식점 매출에 영향을 미치는 외부 요소를 분석한 데이터 분석 프로젝트입니다.

<br>

## 주제 선정 이유

생활권 주변 음식점을 보면 잘 될 것 같은 가게가 금방 폐업하거나, 반대로 오래 살아남는 경우를 자주 목격했습니다.  
결국 **매출이 가게의 생명을 좌우**한다고 생각하여, 매출과 관련된 외부 요소가 무엇인지 데이터로 검증해보고자 이 주제를 선정했습니다.

<br>

## 가설

| 번호 | 가설 |
|------|------|
| 1-1 | 저녁 시간대(17~24시)의 음식점 매출은 **상주 인구 수**에 비례할 것이다 |
| 1-2 | 점심 시간대(11~17시)의 음식점 매출은 **유동 인구 수**에 비례할 것이다 |
| 2   | 주민들의 **소득**이 높을수록 음식점 매출이 높을 것이다 |
| 3   | 음식점 매출이 높은 지역일수록 **카페 매출**도 높을 것이다 |

<br>

## 데이터 출처

[서울시 열린 데이터 광장](http://data.seoul.go.kr/) — 골목상권 분석 서비스

| 파일명 | 사용 가설 |
|--------|-----------|
| 서울시 상권분석서비스(추정매출-자치구).csv | 1-1, 1-2, 2, 3 |
| 서울시 상권분석서비스(상주인구-자치구).csv | 1-1 |
| 서울시 상권분석서비스(길단위인구-자치구).csv | 1-2 |
| 서울시 상권분석서비스(소득소비-자치구).csv | 2 |

> 분석 기간: **2023년 3분기 ~ 2024년 2분기** (최근 1년, 분기별 약 1,540개 데이터)

<br>


## 사용 기술

Python, Pandas, NumPy, Matplotlib, Seaborn

<br>


## 분석 방법

- 자치구별로 데이터를 집계하여 `{자치구: 값}` 형태의 딕셔너리로 가공
- 두 변수 간의 비례 관계 파악을 위해 **산점도(Scatter Plot)** 및 **추세선** 활용
- 음식점 업종은 서비스 업종 코드(`CS100001 ~ CS100009`)로 필터링 (카페 `CS100010` 제외)

<br>


## 분석 결과

| 가설 | 결과 | 해석 |
|------|------|------|
| 1-1 저녁 매출 ↔ 상주 인구 | 상관관계 없음 | 추세선 기울기가 작고 데이터 분산이 큼 |
| 1-2 점심 매출 ↔ 유동 인구 | 상관관계 없음 | 가설 1-1보다 기울기는 크나 분산이 여전히 큼 |
| 2 음식점 매출 ↔ 소득 수준 | 정비례 관계 | 추세선에 데이터가 충분히 근접 |
| 3 음식점 매출 ↔ 카페 매출 | 강한 정비례 관계 | 가설 2보다 더 높은 상관관계 |

<br>


## 결론

1. **인구 수와 매출은 무관** — 인구가 많다고 외식을 더 하는 것은 아니며, 가족 단위 거주 지역은 오히려 집밥 선호도가 높을 수 있음
2. **소득과 매출은 비례** — 소득이 높을수록 식비 지출이 늘어 음식점 매출 상승으로 이어짐
3. **음식점·카페 매출은 함께 움직임** — 식사 후 카페를 방문하는 문화로 인해 두 업종의 매출이 연동됨

<br>
